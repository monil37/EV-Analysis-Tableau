# Electric Vehicle Analysis using Tableau

Implemented a Dashboard to analyse distribution of Electric Vehicles based on Car type in Tableau.


**<p align="center"> Sales Dashboard Preview </p>**

![Sales Dashboard Preview](./EV%20Dashboard.png)


**Procedure**
1. Clone the repository using git command or by downloading ZIP file <br>
` git clone repository-url`
2. Open the tableau workbook named *EV Analysis.twb* which contains detail analysis based on Year, Map, Top 10 Counties including Final consolidated Dashboard.

> [!Note]
> While opening tableau workbook please make sure that dataset *Electric_Vehicle_Population_Data.xlsx.csv* is also in the same directory as workbook
